Стол-трансформер

Описание товара

Стол трансформер для защиты от нападения десептиконов

Характиерстики товара

  • Ширина : 80
  • Глубина : 60
  • Высота : 90
  • +5 к безопасности помещения
  • +10 к неожиданному изменению обстановки

Подробное описание товара

Стол предназначен для спокойного чтения газет и журналов. Однако, при должном старании и танцам с бубном вокруг способен трансофрмироваться в обеденный стол, с помощью которого можно накормить ораву голодных спиногрызов.
